# Secure AWS VPC & IAM Architecture

## Overview
This repository showcases a secure AWS cloud architecture designed
according to AWS best practices and the Well-Architected Framework.

## Architecture Diagram
![AWS Architecture](diagrams/secure-aws-vpc-architecture(1).png)

## Features
- Multi-AZ VPC design
- Public and private subnet separation
- Bastion host access pattern
- Least-privilege IAM roles
- Centralized logging and monitoring

## IAM Policies
- EC2 instance role with minimal permissions
- Read-only auditor role
